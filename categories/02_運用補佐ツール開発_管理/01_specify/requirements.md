## 1. 依頼概要
- 依頼タイプ: 新規提案
- 依頼タイトル: 自分の AI Ops 展開に関するタスク管理Webを作りたい
- 依頼本文: React + Vite の Web UI と SQLite を利用し、顧客名、現状ステータス、反応具合、次アクションを登録・閲覧できるローカル動作アプリを用意する。

## 2. 背景（Why）
- 現在、この PC 上で運用できるタスク管理 UI が存在しない。
- お客様からの相談件数が増えており、案件の抜け漏れ防止のために一元管理が必要。
- お客様への紹介に向け、短期間で見せられる形の成果物が必要。

## 3. 目的（What）
- ローカル環境で起動できる AI Ops 用のタスク管理 Web アプリを構築する。
- タスク情報の登録・一覧・更新・状態把握を可能にし、運用補佐業務の効率を上げる。

## 4. スコープ
### 4.1 実装対象
- React + Vite フロントエンド
- SQLite 永続化
- データ項目: 顧客名 / 現状ステータス / 反応具合 / 次アクション
- ローカル起動手順（Copilot Chat で指示可能）
- テスト実装とパス確認

### 4.2 非対象
- クラウド公開
- 本番運用監視
- 多要素認証などの高度セキュリティ実装
